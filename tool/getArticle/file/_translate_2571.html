　　Microgaming所开发的赌场游戏软件中最流行的是古墓丽影视频老虎机。这个名字听起来是不是很熟悉？当游戏情节与相应的卷轴屏幕符号相结合时看起来更加熟悉，游戏主题与情节与电影明星劳拉安吉丽娜朱莉主演的电影《古墓丽影》一致，这部电影得到了Microgaming的独家授权，并收获了很高的票房，反过来，这部电影使古墓丽影视频老虎机游戏变得更加流行。

　　古墓丽影视频老虎机游戏遵循劳拉冒险精神，劳拉试图寻找难以捉摸的“偶像”的美丽四肢，以展现劳拉自身的超级辣味。该视频游戏还提供一个引人入胜的电影预告片下载。

　　“古墓丽影”在线视频老虎机结构设置有五个卷轴以及十五条赢钱水准线，玩家的投注范围较宽，甚至还允许像一分钱硬币所下的小赌注。在意义上说，它是第一个允许玩家自由旋转奖金条目的网上赌场。

　　一个野生符号（代表古墓丽影的标志），一个分散符号（代表劳拉）和一个难以捉摸的“偶像”（代表你的游戏搭档），达到五个野生符合即可中头奖，达到三个或更多的分散符号可以获得3倍的奖金，三个或更多的“偶像”符号可以进入一个新奖金回合。此奖金回合，又称为“你的坟墓”。

　　此外，初始奖金还增加了与两个或更多的劳拉符号组合，比如两个劳拉符号将双倍奖金，三个劳拉符号将支付4倍奖金，而五个劳拉符号将获得最大奖金支付。

　　正如你可以看到，“古墓丽影”视频老虎机游戏是一个精彩的主题游戏，但玩家们在开始游戏之前请务必详细阅读支付表上的文字内容以及熟悉中奖游戏符号。
