爱尔兰可能很快接受赌场赌博

作者: 博之道 来源: 未知 时间: 2011-11-11 阅读: 次 博彩译文

　　在过去的十年中，赌场赌博已经被几十个国家相继接受，现在看来，即使是爱尔兰可能最后也会接受赌场赌博这样的行为，会将其合法化。爱尔兰司法署已建议在爱尔兰建设具有拉斯维加斯风格的赌场。

　　一份报告指出，前往美国的爱尔兰居民大多数是去拉斯维加斯赌场赌博，从而导致国家对赌博政策的改变。爱尔兰数以百万计的人正在另一个国家用美元来进行赌博娱乐，律政司要改变这样的趋势。

　　在此之前，现行法律中的任何改变，都需要在3月份的换届选举中实行。据司法部长德莫特·埃亨所述，由于有外部因素的影响，此次现行法律的改变可能会提前。有一点是可以肯定的，进行这样的改变，将会使得数百万美元注入当地经济。

　　近年来，许多国家在遵守欧盟法律的基础上已经更改了他们的赌博法律，以此来增加他们的政府收入。在游戏行业，规律变化的结果，就是形成一个世界性的赌博热潮。

　　只要规范赌博的一些规则，大型赌场是允许设立的，法律上的规定至少存在这样的可能性。发展一个拉斯维加斯式的赌场，可能没有那么大的管辖权。虽然他们认为设立两个或三个超级赌场是最为合适的选择，但实际上政府尚未决定将会允许设立多少个赌场。

　　在这样一个赌场“度假”可以得到最终放行之前，所有的都是处于计划的过程中，政府和相关机构的审核仍需通过。
