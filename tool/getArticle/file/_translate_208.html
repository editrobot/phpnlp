百家乐小强路记录牌局发展介绍

作者: 博之道 来源: 未知 时间: 2011-09-05 阅读: 次 博彩译文

　　玩百家乐，想要多点胜算，就是要把它的牌局发展记录下来，这种就做写路，一般可以分为：珠盘路、大路、大眼仔路、小路、小强路等等.下面博之道就为大家介绍一下小强路的记录法吧。

　　百家乐小强路推算的起始位置从第二行第四粒写起，比推算大眼仔推后了两格。推算时需要向前看（向左看），然后跳两格，跳完格后，看之前是否有大路，有的话，小路用红，没有的话，小路用蓝。同样，直排的第一行写法比较特殊，向前看，看所在格之前的前四列，在这四列中需要跳过中间的两列，即要看的两列是与所跳列相邻的两列，看这两列是否齐脚，如果齐脚，小路用红色，反之则用蓝色。特别需要注意的是，在推算时，在每一列大路结果中，除去参照列（推算时您所要看的列）第一个没有大路的情况，小强路用蓝色外，该列之后的小强路都应该用红色。

　　为了加深您的理解，我们以下面的大路计分卡为例来推算小强路。

　　1. 从大路的第二行第四粒写起，即从11开始，向前看，跳两格，即跳过大路6、8，看到大路2，小强路用红色；

　　2. 到12号，向前看，跳过9和空格，前面有大路3，小强路用红色；

　　3. 到13号，向前看，跳过两个空格，前面有大路4，小强路用红色；

　　4. 到14号，向前看，跳过两个空格，前面没有大路，参照列第一次出现了没有大路的情况，小强路用蓝色；其后的该列的大路15，小强路用红色；

　　5. 到16号，16所处的位置是直排的第一行，写法比较特殊，向前看16所在列之前的前四列，跳过这四列中间的两列，即跳过5所在列和7所在列，看10所在列和1所在列。10所在列有6粒大路，而1所在列有4粒大路，很显然不齐脚，小强路用蓝色；

　　6. 到17号，向前看，跳过11和8，前面有大路6，所以小强路应该用红色；

　　7. 到18号，向前看，跳过10所在列和7所在列，看16所在列和5所在列。此时您可以看到16所在列有2粒大路，而1所在列同样有2粒大路，显然齐脚，小强路用红色；
