　　五张牌规则（一）

　　五张牌，也叫加勒比海扑克，是广受欢迎的一种澳门百家乐赌场Casino扑克澳门真钱百家乐游戏。在五张牌澳门真钱百家乐游戏里，网上真钱百家乐玩家只需与庄（Banker）斗大，而不是网上真钱百家乐玩家间互相对垒，赢钱将给手中握有较高牌的一方。网上真钱百家乐玩家首先将赌注置于赌桌上。网上真钱百家乐玩家发到五张牌，庄（Banker）也拿到四张面朝下和一张面朝上的牌。然后网上真钱百家乐玩家评估手上的牌并作出以下其中一个选择。

　　1.投降 - 网上真钱百家乐玩家将会失去赌注。

　　2.加注 - 网上真钱百家乐玩家把原赌注两倍的筹码放到加注区域。

　　如果澳门真钱百家乐游戏还能连续，庄（Banker）便会翻开手中的牌并比较网上真钱百家乐玩家的牌。然而，庄（Banker）必须有一张A和一张K或更好的牌才算够格。如果庄（Banker）不够格，网上真钱百家乐玩家便赢得两倍原赌注，及一倍加注区赌注。如果庄（Banker）至少有一张A和一张K而够格，同时牌又高于网上真钱百家乐玩家的牌，网上真钱百家乐玩家便输掉原赌注及加注的赌金（即头注的三倍金额）。如果你的牌与庄（Banker）平手，所有澳门百家乐投注退还。如网上真钱百家乐玩家的牌高于庄（Banker），网上真钱百家乐玩家便赢得与原赌注相等的赌金并根据下表获得加注部份的红利。

　　牌的搭配加注部份的红利皇家同花顺100倍同花顺50倍四同张20倍葫芦（三带二）7倍同花5倍顺子4倍三同张3倍二对2倍一对或高牌1倍

　　此外，五张牌还有累积奖金，网上真钱百家乐玩家有机会赢得累积奖池，不过需要下额外的1元边注才有机会赢，累积奖金比较多时可以去碰碰运气。有一点需要注意，获得累积奖金不需要庄（Banker）够格，只要你拿到好牌并且下了边注就可以获得奖金，举个例子，你拿到同花，而且下了边注，那么就可获得50倍的奖金，即使庄（Banker）的牌比AK小也没关系，同理，拿到四同张可得500倍，拿到同花大顺即可得到整个累积奖池。
