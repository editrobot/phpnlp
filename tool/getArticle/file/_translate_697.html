赌场上闪耀的明星―克里斯弗格森

作者: 博之道 来源: 未知 时间: 2011-10-13 阅读: 次 博彩译文

　　克里斯弗格森是扑克游戏上最大的明星。他的长头发，胡须和牛仔帽的独特打扮，克里斯弗格森成为容易的辨认、专业扑克的知名的玩家。除了从他的不寻常的外表装扮，克里斯弗格森给人带来的还有独特的素质和在主机游戏中别树一帜的能力。

　　扑克名人克里斯弗格森及其职业

　　克里斯弗格森打扑克的兴趣是在他对计算机科学博士学位的研究期间引发的。他利用自己的知识和他父亲的数学博弈论的知识，使得弗格森在1994年征服了扑克世界。

　　尽管弗格森的数学背景优势是经历一个缓慢的开始在他的职业生涯中，在金钱上来说，1995年和2000年之间他一共进行了16个场合，但未能赢得一个单一的比赛，输掉不少钱。

　　经过5年的艰苦努力，克里斯弗格森夺取了他的第一场比赛冠军，在2000年赢得世界扑克系列赛的7张牌梭哈限制比赛和采摘151 000美元奖金。两个星期后，克里斯弗格森又中奖，在无限额德州扑克锦标赛中赢得比赛的重头戏，口袋里多了150万美元的一等奖。

　　次年，克里斯弗格森赢得奥马哈高低分割八或更好的比赛，在2003世界扑克系列比赛中，这可谓是一个壮举。2003年世界系列扑克中，克里斯弗格森反复证明了他的技能和通用性也有所回升，半限制的标题保持半限位螺栓，使他的世界系列扑克锦标赛的冠军数增加到5个。

　　虽然克里斯弗格森在他的职业生涯已经赢得了超过一百万美元，他玩牌的强度增加，在拉斯维加斯几个主要扑克锦标赛赢得一个大奖，包括一个在2008年的扑克锦标赛的50万美元的奖项。

　　目前克里斯弗格森的奖金超过600万美元，这个伟大的玩家，他的名字呈现在11场比赛的胜利者中，而每次只需100元现金就可完成。

　　克里斯弗格森简介和玩牌风格

　　克里斯弗格森大量数学和博弈论来打扑克。他声称，研究计算机程序扑克的模型和依靠他的设计，以协助他完善自己的战略。弗格森也有利于积极的玩牌风格，会让他的对手置于恒定的压力下。
