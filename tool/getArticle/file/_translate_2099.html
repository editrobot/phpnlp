　　对于许多彩票痴迷者来说，西班牙超级彩票厄尔尼诺戈多是今年最大的事件。目前厄尔尼诺戈的彩票已经开始销售了，它是世界上奖金最丰厚的彩票游戏之一，今年预计将突2.5亿美元。

　　2011年的厄尔尼诺戈多超级彩票也许是有史以来奖金最丰厚的一次，它在12月22日制定出来的。整个西班牙都在为这个超级彩票而狂热，不少西班牙人甚至花费几个小时排队购买该彩票，而抽奖的正式名称正是“厄尔尼诺戈多”。

　　有一个令人难以置信数据表明，西班牙98％的成年人购买了“厄尔尼诺戈多”彩票，此外还有来自世界各地的成千上网的玩家也在法网上购买该彩票。

　　面对“厄尔尼诺戈多”超级彩票的巨额奖金诱惑，许多投注者，尤其是在西班牙投注者，都使用群体投购买模式，以降低成本，增加夺冠赔率。
