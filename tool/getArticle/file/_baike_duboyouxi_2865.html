骰宝规则：真人骰宝规则介绍

作者: 来源: 互联网 时间: 2011-12-14 阅读: 次 赌博游戏

　　骰宝，又被称为赌大小，英文叫做Sci Bo，是民间一种古老的使用骰子作为赌具的赌博游戏。骰宝游戏在民间拥有很广泛的群众基础，真人斗地主有不少人喜欢在闲暇之余玩玩骰宝游戏。随着网络科技的发展，出现了真人骰宝游戏。真人骰宝规则与民间古老的骰宝规则基本一致，也就是由各闲家向庄家下注。每次下注前，庄家先把三颗骰子放在有盖的器皿内摇晃。当各闲家下注完毕，庄家便打开器皿并派彩。因为最常见的赌注是买骰子点数的大小（总点数为4至10称作小，11至17为大，围骰除外），故也常被称为买大小（Tai-Sai）。

　　真人骰宝规则具体如下：

　　1. 骰宝游戏采用三粒骰子， 客户可在投注时间内下注投注项目

　　2. 投注时间结束後， 三粒骰子在自动振动骰盅内进行滚动。 当振动时间完毕後， 静止的三粒骰子朝上一方的点数便是该局的结果

　　3. 因斜骰或叠骰…等因素令结果不能清析判定。 监场人员有权把该次结果设为无效。 该局投注会按重新摇骰後的清析结果计算

　　4. 当游戏结果出现围骰， 投注（大／小 ）或（单／双）项目均作为输计算

　　骰宝是庄家永远处于有利位置的赌博游戏。足球盘口闲家无法以技术提高得胜的机会，长远来说庄家必胜。各种投注中以「大、小」对闲家最为有利，但庄家仍然拥有优势。
