如何禁止赌场作弊？

作者: 博之道 来源: 未知 时间: 2011-10-17 阅读: 次 博彩译文

　　这是一个有意义的问题。显然有人喜欢对此进行讨论！有人会问：“在英国，如何禁止赌场作弊？”

　　我们假定，这种作弊的人不希望得到禁止，而是需要知道他作弊多久，直到被揭穿，被送上监狱。换言之，他要被赌场列入黑名单。

　　因此，我们必须就此事进行研究。下面，是我们的一些发现：

　　让我们先从二十一点游戏开始，因为这是历史上受大多数人喜爱但已被禁止的游戏。似乎总是有人试图占据对他们有利的赔率：

　　这里有一些活动，在赌场外面，你总会看到的：

　　算牌。二十一点算牌开始于麻省理工学院的一个团伙，至今它仍在继续，因为这可以提高您的赔率，特别是对单层游戏。如今甚至有一个iPod的应用程序，在你的口袋里震动帮你算牌。但不要抓到，否则你会列到黑名单（甚至可能被捕）。

　　二十一点表以及它的一些技巧（记得到处都有CCTV相机）：

　　作为一个团队，让你的伙伴从经销商背后的偷看卡的信息，然后给你发信号。

　　将您的赌注添加到已经处理的后一轮的筹码。

　　使用电子计数卡，如iPod。

　　你要防止喝得太醉。在玩赌场游戏时，最好不要喝酒，酒是你的紧箍咒。你认为他们为什么会提供免费的啤酒？如果要喝，可以，但是，是有一定限度，至少要保持你的头脑清醒。

　　轮盘赌：

　　你要禁止“张贴过去”。这是作弊的一种技术，用在赌场是非法的。

　　在这个骗局，诈骗者通常有魔术师般快捷的动作。通常该团伙的团队巧妙地分散赌场的各个位置上，方便转移筹码，所以它不是太明显。然后，他们会打几个标准的旋转然后切换表或换赌场。它可以是非常微妙的，很难发现，但事后分析赌场号码的人就会明显，它往往是采取“赢了就跑”的战略 - 一个玩家在赌场打一个车轮，然后转移筹码。这种行为是高度非法的，它基本上是一种偷窃行为。

　　另一个版本是顶制帽，这是像“张贴过去”，但经销商是设骗局的一方。那么，什么是顶制帽？这也是高度的非法行。

　　赌场经销商使用灵巧的花招把中奖号码藏在口袋里。这可以相当有利可图的。

　　不过，一般情况下，罪犯的贪婪都会被发现。 （但是，我们从来没有听到关于一个人的消失，理解了，赌场不敏锐，宣传这种事情）。

　　曾经有一个比较有名的赌场欺骗了许多玩家，是在大西洋城的由理查德马库斯和蒙特卡罗经营了25年的价值500万美元的赌场。
