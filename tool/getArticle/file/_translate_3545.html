谢尔登・阿德尔森，一个喜爱冒险的企业家

作者: 来源: 时间: 2012-01-05 阅读: 次 博彩译文

　　谢尔登·阿德尔森是一个自我成就，喜爱冒险的波士顿企业家。在1979年举办的世界计算机博览会取得了巨大的成功之后，他把维加斯变成了顶级的会议中心。他是一个活跃而又超级保守的人，他成功地，也似乎是故意地把其他赌场大亨的敌人们变成了一个友好的小团体。最近，他的敌人温恩邀请他参加一起吃饭，化解彼此恩怨，他拒绝了。原因是温恩曾经有一次把他叫做“脱线先生”（译注：脑子不好使的意思）。阿德尔森在会议室接受采访的时候，他会不停地打断谈话，提醒记录他生平的录影员换带子或者更换角度。

　　但是除了确保自己在镜头前面展示好的一面之外，阿德尔森也不见得担心很多事情。他担心自己的锻炼和节食计划，以减小自己的啤酒肚。“报纸上说我们一只脚踏入了破产，另外一只脚踩在香蕉皮上，头上还吹着大风。如果你相信这些，那就大错特错了。”他说。阿德尔森总是相信自己，他已经在自己的公司里面追加了10亿美元投资。但是他也解雇了自己被股东起诉的长期战友，并且自11月份以来，他还解雇了700多名拉斯维加斯员工。

　　他看上去没有被自己的损失压垮。“10亿美元买不回过去。事情别人想象的那么糟糕，”他说。“我对我妻子说，生意上最坏的事情会让我哭两小时，但我现在一分钟也没哭过。”当他妻子要求他缩减开支的时候，他拒绝了，告诉她他手头的钱还够花。最终他还是屈服了，那就是在可能的时候，使用自己的小飞机，而不是大飞机。

　　阿德尔森也许已经更换了飞机，但是他还是没有改变使用高规格饮食、巨大的套房以及豪华的会议空间吸引消费者的策略。他不相信美国人会根本性地改变价值观，从奢华走向节约。“这个世界不会改变，人们不会停止自己想做的事情。人们不会说，‘我支持旧习俗，去加州看红杉林，而不是去拉斯维加斯；或者说我会带着书去鳕鱼角。’我不认为他们会那么做。我习惯这个圈子，我不想回头，”他说。“这是人的本性。就像老歌唱的那样，一旦人们到了热闹的巴黎，哪还能安心地呆在农场？”阿德尔森的银行账户已经被衰退扫荡的差不多了，但他最后一段唱的还很不错。

　　但是我知道，在这所有的自信背后，都有一些深刻的教训。他在这次衰退中得到的教训比任何人都深刻，那就是不能借自己无法偿还的钱。他的债务即使是情况好转也很难还清。因此当我问还需要多久才考虑得到贷款进行再次扩张的时候，我期待他为自己的鲁莽而道歉，并且会允诺自己会成为储蓄者。他站起来，睁大眼睛并微笑着说，“只要有人愿意借，我立马就接受。”

　　因此维加斯正在进行自己的赌局。维加斯人坚信这只是另外一个商业循环。衰退很快就会结束，并且人们将会再次回到老虎机上进行赌博，地产市场会再次繁荣，试吃菜单价格会再次上扬，企业也会再次得到两位数的利润。事实上，温恩和我打了100美元的赌，我花了几分钟才让他明白这个数字的意思，赌的就是美国的GDP将会在2011年4月实现增长。同时，他和其他维加斯大亨相信人员会被重新洗牌，有另外的人会成为赌场的主人，但是游戏自身并不会改变。他们认为美国人会继续变的更加富有。看着这座沙漠城市什么都没有创造出来，现在徒留下一座空荡荡的鬼城的时候，就觉得这种说法不切实际。很多人已经离开，逃离了这个赌局。除此之外，拉斯维加斯人别无选择，只能赌所有的事情都会回来。他们每个人都身在其中。
