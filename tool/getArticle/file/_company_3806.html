2012年度世界博彩公司50强名单新鲜出炉（三）

作者: 博之道 来源: 时间: 2012-01-20 阅读: 次 博彩公司

　　21.Interwetten

　　Interwetten 公司是新兴的欧洲体育博彩业中不可忽略的一家公司，然而由于该公司产品缺乏多样性从而导致其排名有所下滑。该公司拥有德国颂发的许可证，占有举足轻重的战略位置，其博彩场合理分布在欧洲市场中，并且拥有丰富的经验。最近，该公司进行重新部署，移兵马耳他，这不失为明智之举，我们仍把该公司当成一个重要角色。

　　22.SportInteraction

　　SportsInteraction 公司将其重心放在北美博彩业上，由于该公司经营涉猎有限，成为大角色的可能较小，其影响力不足以道。然而，其拥有巨大的规模，而且业绩也已超过其所能及的对手。最近，该公司将 Playtech 纸牌博彩添加至其赌场后，为其增收扩大了渠道，从而有可能向新的台阶跨越。

　　23.Golden Palace

　　我们无法获知 Golden Palace 公司的赌场及纸博博彩赢利数据情况，但我们认为该公司处于赢利状态。然而，将该公司首次带入前25名的并非其雄厚的经济实力。过去的12月里，Golden Palace 公司将游击式营销方式推向新的高度，设法保证其品牌始终为大众在线彩民所知。其所带来的广泛影响力和产品多样性使得该公司首度跻身前25名。

　　24.Empire Online

　　尽管受 PartyGaming 的实力所影响，身为 Empire Poker 公司之母公司，该公司筹资2.25亿美元成功上市仍不失为一件大事。该公司在2005年第1季度赢利1250万美元，其 Empire Poker 的品牌是 PartyPoker 网络中最大的子公司。然而，其局限于白标运营商的地位以及缺乏产品多样化使得它只能在排名中相对靠后。

　　25.Gaming VC

　　Gaming VC 公司作为唯一的在线上市博彩公司，在评估业内各公司的实力时，该公司仍在圈中外围徘徊。由于该公司缺乏德国外的收入，这使得其影响力流于中庸。若来年它仍能作为一条吃鱼的鱼而不是等待被吃的鱼的话，那么它就有望于到时重振士气。

　　26.Fun Technologies

　　Fun Technologies 公司正在将技能博彩在在线博彩业中的地位往回推，很难说其在该新兴行业中的影响力到底如何。不像其诸多对手，该公司在地域上分布广泛，经营产品多样，其在 BetandWin 为主要股东的 Betbull 公司中拥有大股，在投注博彩业中始终占有一席之地。目前，Fun 公司所缺的是利润。

　　27. eWorld Holdings

　　eWorld Holdings 公司作为纸牌博彩公司五强之一，旗下拥有 UltimateBet 公司，该公司正将其影响力从美国大本营向欧洲延伸。其营销业绩已更上一个台阶，该公司目前正盯准美国和英国的大型离线活动。本年度下半年该公司将增开一家赌场，从而成为新增利润的支柱。

　　28.Mr Bookmaker

　　Mr Bookmaker 公司位于马耳他，经营体育博彩和赌场，在欧洲法语地区有较大影响力，这主要是因为该地区对各种赛事的赞助之故。作为能有效锁定法国新兴市场的少数几家运营商之一，而且身兼欧洲博彩协会之主席，该公司影响力较为深远。应紧密关注。

　　29.Fortune Lounge

　　Fortune Lounge 公司这家强大的赌场集团最近将其营销业绩又推进了一步，其在美国的媒体投入可与其他赌场巨头相比论。该公司业绩稳定，有传言称其有可能要上市，再加上其迅速扩展的纸牌博彩场，这些将可能令其2006年大出风头。

　　30. Skybet

　　Skybet 公司作为电视巨头 BskyB 公司的独有子公司，仍为英国 iTV 业中最具实力的公司，它对英国以外的市场概不过问，同样使其以前无缘前30强，但考虑到其巨大的营业利润，该公司仍是一支劲敌。
