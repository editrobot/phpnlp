澳大利亚体育博彩的兴起

作者: 博之道 来源: 时间: 2012-03-22 阅读: 次 博彩译文

　　澳大利亚国会议员们在过去一年里曾试图起草法律文案，以限制国民在赌场上的赌博金额，但现在他们发现另一种形式的赌博正在席卷澳大利亚。

　　体育博彩在世界各地的许多国家是属于非法的。但是在澳大利亚，体育博彩越来越受欢迎。在过去的五年里，澳大利亚的体育博彩增长了近15％，这个数字令一些当地人感到十分担忧，他们希望政府能加强这方面的监管。

　　德怀特·兰斯甚至抨击国会议员，说：“他们想制定一些法律条文来限制国民在赌场上的赌博金额，但这样的行为其实是在浪费他们自己的时间。随着体育博彩的人数不断增加，他们应该关注的是这些人是不是有可能会失去他们毕生的积蓄，他们的工作，以及他们的家庭。政府应当设法控制这样的局面。

　　到2012年底，分析家预测，澳大利亚人每年投注的赌金高达225亿美元，这一数字只有少数几个国家能与之相提并论。澳大利亚体育博彩的崛起得益于网络体育博彩，赌客不用离开自己的家，就可以在网上进行体育赛事投注。

　　澳大利亚IBISWorld公司的总经理凯伦Dobie说：“网上体育投注是最方便的赌博形式，而舒适的网络环境是推动体育博彩经济增长的一个重要因素。”

　　足球投注是举世公认的体育博彩。在澳大利亚，足球是全国性的娱乐项目，赌球也已经成为许多澳大利亚人一种广泛的爱好。平均而言，澳大利亚投注总额一年高达1256美元，这一数字只有英国和美国能与之匹敌。
