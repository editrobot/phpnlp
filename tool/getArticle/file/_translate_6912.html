欢乐岛的委员们再次考虑赌场博彩项目

作者: 博之道 来源: 时间: 2012-06-24 阅读: 次 博彩译文

　　在四月份他们曾经对赌场博彩说不，但是现在欢乐岛的委员们再一次把目光投向赌场博彩项目。

　　在4月3日，委员们以4比2 的投票结果否决了出售土地期权给阿瑟港复兴集团的提议。

　　阿瑟港复兴集团收购的土地期权是为了在欢乐岛修建一处赌场度假村。

　　该集团需要土地期权是为在阿瑟港发展赌场博彩项目创造条件。

　　今晚，欢乐岛的委员们召开秘密会议，但是没有作出决定。

　　据悉他们要求律师与复兴集团涉及相关协议的律师艾伦.派克进行会晤。

　　这项协议将在7月10日的一次会议上进行表决。欢乐岛之所以会再次考虑这个问题是因为获悉福特公园正在这方面伺机而动。

　　福克斯新闻频道获悉复兴集团正在与杰佛逊县官员进行会谈，商议如果德克萨斯州批准赌场博彩的话，在福特公园兴建赌场博彩项目。

　　有消息说，复兴集团将考虑接手福特公园娱乐部分的债权债务。
