天下四分

作者: 吴有根 来源: 环球博彩 时间: 2011-12-16 阅读: 次 博彩杂志

　　吴有根是澳门倍受尊重的扑克玩家，通晓有关澳门扑克的话题。《环球博彩》很荣幸出版他关于澳门扑克业历史、现状与未来的独特视角。

　　目前，德州扑克在澳门的发展方兴未艾。我们在此介绍下澳门的 扑克室发展状况，以及各自的风格。澳门现有四间扑克室，分别 为新葡京的扑克之星、永利扑克室、星际扑克王俱乐部以及威尼 斯人扑克室，这个数字还在增长中。

　　威尼斯人星际原本是第一家提供扑克游戏的赌场， 早在2007年8月，就已经开设了5张电脑发牌的 扑克游戏桌PokerPro。那时候，星际是唯一能 够玩德州扑克的去处，可谓是独市生意，周末 期间全部爆满。可惜其后知后觉，没有把握 及早开设人工发牌扑克室，以至被新葡京后 来居上。其电脑发牌的扑克室最终在次年关 门大吉。

　　澳门首场扑克赛事是2007年11月在新葡 京赌场举办的PokerStars.net亚太扑克巡回赛 (APPT)。

　　新葡京拥有澳门第一家提供现场人工发 牌的扑克室，开业于2008年2月。一开始就非 常火爆，周末的时候八张台全满，排队的玩家 众多。由于扑克室的位置靠近三楼中央，毗邻 观看赛马的场地，以至吸引不少其他游戏的玩 客尝试扑克。


威尼斯

　　澳门扑克之星于2008年5月在金都娱乐场 开设了人工发牌的扑克室。也许是地理位置 原因，到访的玩客并不多。那时候，扑克在澳 门可谓多水多汁，只要有点经验，起手牌打得 紧点，就足够盈利。这样的情况持续了一段时 期，经过口碑载道，来自世界四面八方的职业 玩家纷纷来到，令新葡京的扑克室如日中天； 然而这种垄断的日子没有持续。永利扑克室于 同年11月开业，新葡京的生意突然慢下来。2009 年3月澳门扑克之星搬迁并融入到新葡京的扑克 运作中。为了对抗永利的竞争，新葡京实施了现 金与比赛的双路线并轨，成为其专有的特色。

　　目前，新葡京已经成为澳门扑克 赛事的主要场地，每年APPT澳门站的比 赛都在此举行，而这场比赛也迅速成为 世界扑克玩家不容错过的赛事；加上较 为本地化的红龙杯，以及每月几十场中 小型比赛，好不热闹。由于有比赛的支 持，现金桌生意也恢复得不俗，平日开 三、四张台；节假日有比赛的时候，十 多张台经常爆满，主要集中在比较低的 (港币)10/20注码。

　　永利是第一家开设扑克室的外资 赌场，开业于2008年11月。由于经验丰 富，以及服务出众，一开始就把新葡京 的生意抢了不少；加上永利的中场及贵 宾厅的大赌客众多，很快就取代了新葡 京的领导地位。一路走来，更成为高注 额玩家的乐园。高峰期的时候，一位难 求。就算在平日的黄金时段，等位两三 个小时也不足为奇。

　　永利决定把注码提升，瞄准高端 市场。基本上10/25的台只有2、3张， 其余的都是25/50、50/100、100/200 及2 0 0 / 4 0 0 ， 而超高的5 0 0 / 1 0 0 0 、 1000/2000及3000/6000都曾经出现过。 这些大额现金赌局经广泛流传，世界一 流的高手开始到访。无需惊讶，那些世 界扑克联赛(WSOP)手链得主以及电视经 常出现的扑克界名人，也到这里掘金。

　　本来，永利可以趁势雄霸澳门的扑 克市场，可惜它扩张缓慢，以至等候时 间太长，把小客都赶走到其他扑克室， 实属憾事。另外，永利目前是第一家 （也是唯一）提供奥马哈（Pot Limit Omaha）游戏的娱乐场，奥马哈将是下 一个风行扑克界的游戏，因为其冒险性 满足了当地市场的需要。如果您钟情奥 马哈，就必须到访永利。

　　星际最终也开设了人工发牌扑克 室，定名APT扑克室，被韩国人承包。可 惜为时已晚，尽管亚洲扑克巡回赛(APT) 澳门站的比赛每年在这里举办，却始终 未能打开局面，最后易手。

　　澳门星际赌场扑克王大赛辗转之间，星际的 扑克室由大型赌厅太阳城 承包，更名为扑克王俱乐 部。借着电影《扑克王》 、杂志、扑克网站等的大 型攻势，于2009年10月末 开幕，誓必一争天下。由 于营运者的背景，星际 很快便已夺取超大额注 码的市场，目前200/400 、500/1000及更高的注 码，多在星际开台。这 里还提供很多小额扑克 桌，玩客同样享受周到 服务。

　　值得一提的是，星 际的营运者可谓不惜工 本，誓言争夺市场。只 要加入会员，不但可以 累积积分，以换取礼物和 餐券；更有每两个小时的 幸运抽奖、每天的最佳牌面组合奖等活 动，以各种优惠政策，吸引众多玩家。 这种情况，可以说是竞争下的产物，是 价格战的另一版本。星际扑克王新的营 运者吸取了教训，在服务、优惠政策等 方面确实下了不少功夫。

　　威尼斯人开设了澳门最新一家扑 克室，紧随星际扑克王俱乐部于12月开 幕。让人费解的是，这在澳门的扑克圈 竟然无人知晓。在没有任何宣传和广告 的情况下，一开始生意自然冷清。他们 还决定在发牌时使用塑料牌盒装牌，这 是一种让人困惑的奇怪做法。

　　澳门威尼斯人然而威尼斯人凭借大型中场的优 势，生意反而渐渐好了起来，后来的结 果令人刮目相看。周末等黄金时段，10 张台可以全开，但平时通常比较安静。 没有星际式的优惠，没有永利式 的服务，也没有澳门扑克之星的经验，威尼斯人依靠其大场的优势，仍然很快 找到了其市场空间。很多职业选手蜂拥 而上，很快就把扑克室挤满。威尼斯人 扑克室的奇迹，说明了中场支持的重要 性。只要中场大、玩客多，就不愁吸引 到足够的职业、半职业的玩家。目前， 威尼斯人集中于小注额的10/25桌，也 有25/50及50/100的游戏。

　　跟注？加注？反加注？还是弃牌？ 四分天下之后，恶斗依然持续，最终谁 将统领中原？新葡京和永利发展都已比 较健全，显然是目前市场的领先者；扑 克王在迅速完善自己，并已经开始占领 一席之地；威尼斯人与前面三个相比还 有些差距，不过如果产品对路，它却有 其独特的长期优势。结局如何，一切还 得看各自的管理层如何出牌。（2010年3月/4月）
