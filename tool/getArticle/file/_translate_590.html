　　处理所有的卡后，您立即得知您是否已经赢得了比赛。但如何确定自己是否是软件的赢家呢？这样，百家乐变得有点棘手。还记得你在庄家或玩家的赌注吗？如果您所选择的一方，处理具有较高的总牌，那么你就赢得了比赛。是由所有牌卡的总数加起来然后加第一位数。例如，如果玩家的牌点增加至17，或者，它实际上只是一个7。如果卡的总数为11，那么实际的百家乐总额为1。这就是为什么在百家乐游戏中尽可能高的卡一共是9。所以，如果你下注的庄家或玩家牌点是九时，那么，你就可以赢得了移动百家乐游戏。
