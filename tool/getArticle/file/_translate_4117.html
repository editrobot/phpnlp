2013年北美彩票会议将在罗州举行

作者: 博之道 来源: 时间: 2012-02-11 阅读: 次 博彩译文

　　据当地媒体近日报道，美国罗德岛州被选为2013年“北美国家和省级政府奖券协会”（NorthAmericanAssociationofStateandProvincialLotteries，NASPL）年度会议东道主。“协会在最近的一次会议上决定，在我州举办年度会议。”罗德岛州彩票机构主任杰拉尔德·奥宾（GeraldAubin）说。

　　据了解，“北美国家和省级政府奖券协会”年度会议举办时间为2013年9月29日至2013年10月3日，与会者人数预计达到1000人，所需酒店房间数目将超过600个，展台区域面积将达到4万平方英尺。“这一活动将会带给我州至少100万美元的经济效益，当然，这100万美元不包括会展中心和酒店的收入。”

　　“北美国家和省级政府奖券协会”的成员包括美国52个州际彩票机构、加拿大、墨西哥和波多黎各。
