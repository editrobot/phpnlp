安徽四人斗地主必胜攻略教你轻松玩可以赚钱的斗地主

作者: luoluo 来源: 时间: 2012-08-08 阅读: 次 赌博游戏

　　我所知道的四人斗地主游戏玩法里面有宁波玩法，和安徽四人斗地主玩法，但是只是对安徽四人斗地主玩法规则比较熟悉，所以去玩可以赚钱的斗地主游戏的时候我都会选择安徽玩法。我接下来要给大家介绍的就是这种我熟悉的玩法。

　　安徽四人斗地主游戏玩法是采用两副牌和四位玩家的斗地主游戏。没把牌在发片之前都会预先留下8张牌作为底牌，再在剩下的牌当中翻来再插入扑克中，发牌的时候如果哪位玩家拿到这张牌就有优先抢地主的权利。

　　首先从拿到翻过来的那张牌的玩家开始，逆时针方向依次询问是否有人抢地主，抢地主者需要亮出所有牌再拿底牌。如果没人抢，从庄家开始选择是否当地主，若庄家和下家都不当，重新发牌。

　　1．庄家抢地主，输*2赢*3（超级版本：庄家抢地主，输*2赢*4）。

　　2．闲家抢地主，输*2赢*3。

　　3．庄家当地主，输赢*1。

　　4．庄家下家当地主，输赢*1但庄家手上炸弹作废，只可拆开出。

　　5．庄家和下家都不当地主，重新发牌，庄家和下家每人付给其他两家1倍底金。

　　6．当地主后可以选择明牌，明牌后输赢*2。

　　牌型：

　　1．单张：

　　2．对子：可出对大王或对小王。

　　3．连对：三连对或以上。

　　4．三张：

　　5．三张带一对。

　　6．连三张：两个相连的三张或以上，可带同数量的对子，或不带。

　　7．炸弹：

　　大小：大王>小王>2>A>K>Q>……6>5>4>3。

　　牌型比较：炸弹大于其他牌型，其他各牌型之间不可相互比较。

　　炸弹：

　　1．四个王：最大的炸弹，天尊。

　　2．8张：天炸。

　　3．7张：七巧。

　　4．6张：六喜。

　　5．5张：炮。

　　6．4张：枪。

　　炸弹比较首先比较炸弹张数，张数相同的时候比较点数大小。

　　喜金：

　　当有人摸到七巧和七巧以上炸弹并炸出时，其他所有人都要给该玩家相应的喜金。

　　（超级版本：六喜：底金*1）。

　　七巧：底金*1（超级版本：*4）。

　　天尊：底金*1（超级版本：*4）。

　　天炸：底金*2（超级版本：*8）。

　　可以赚钱的斗地主游戏安徽四人斗地主玩法比较特殊的地方有以下的两点：

　　一、当牌数低于10张的时候要报警，告诉其他玩家自己剩下的实际的牌数。

　　二、拿到四个王，但是选择了拆开出的话，就不可以翻倍了。
