　　提示1

　　在投注当中的和局，这是获得金钱的一个诱人的地方。看似宽松的赔付其实是隔靴搔痒，这只是为了弥补命中几率低的缺陷。当一名玩家在和局上下赌注并且赢得比赛，赌场通常会提供8- 1或9到1的赔付。

　　提示2

　　当你去玩百家乐游戏的时候，你应该想到在你的口袋里应该还有一点点额外的现金，当你曾在每一个庄家中打赌赢钱，你就会亏欠庄家5％的佣金。这笔钱并不是马上就由荷官收集。每个玩家欠了多少佣金，荷官会进行跟踪。

　　提示3

　　不要用比你拥有的更多的钱来打赌。如果欠下赌博债务，你将一无所有，这会让你陷入困境。要设置限制，正当你赢了很多钱，到达限制时，要适时离开餐桌，不要以为你可以长胜不输。

　　提示4

　　人们不应该在不合逻辑的赌注上投注。当投注者三次连败，却运用百家乐战略在和局上投注赢钱。要记住，平均有14％赌场优势的和局，将会让你失去所有的资金。

　　提示5

　　百家乐是一种充满着运气的游戏。企图通过投注系统和算牌来赢钱那是在浪费时间。
