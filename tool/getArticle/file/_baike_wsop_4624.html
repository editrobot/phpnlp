法国金秋九月上演第六届世界扑克系列赛

作者: 博之道 来源: 时间: 2012-03-14 阅读: 次 世界扑克锦标赛

　　第6届WSOPE即将连续第2年在法国戛纳的Casino Barrière de Cannes Croisette and the H?tel Majestic Barrière举行。时间安排为9月21日至10月4日，届时将会安排7场金手镯赛事。

　　以下是WSOP副总裁Ty Stewart的讲话：

　　“我们已经迫不及待要在2012年回到Cote D’Azur了。我们把时间表提前了几个星期，这样就能享受到更好的气候以及戛纳能提供的一切。这项德州扑克锦标赛现在已经证实是法国当地重大的锦标赛之一了，这个经历是首屈一指的。现在媒体已经预测会有1,000名选手参加主赛事了，我们会为此做最好的准备。

　　赛程上的金手镯赛事的安排与去年相同。按照WSOP的安排，在系列赛期间会有超过48场边赛。比赛最大的亮点是€10,000 + €450买入金的主赛事，开始时间为9月29日。曾经获得过这场主赛事冠军的选手有Annette Obrestad和John Juanda。

　　赛程表上还有€10,000 + €450买入金的“mixed-max”赛事，比赛第1天为9人桌，第2天改为6人桌，最后一场为单挑桌。

　　以下是金手镯赛事时间表：

　　9月21日，6人桌无限注德州扑克（3天），€2,500 + €200

　　9月22日，无限注德州扑克（5天，第1轮2天），€1,000 + €100

　　9月24日，底池限注奥马哈（3天），€5,000 + €300

　　9月25日，无限注德州扑克Shootout（3天），€3,000 + €250

　　9月26日，Mixed-Max—无限注德州扑克（4天），€10,000 + 450

　　9月27日，6人桌底池限注奥马哈（3天），€1,500 + €150

　　9月29日，主赛事—无限注德州扑克（5天，第1轮2天），€10,000 + €450
