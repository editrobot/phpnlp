　　投注山东体彩山东体彩11选5任选二，如果硬要选择包号，那么，包3个或者4个号码是最佳选择。因为包3个号码复式，1期只投入6元，亏了也不多，一般有中2个号码的可能，是够保本，重要 的是可以坐2望3,这样，就有利可投了。包4个号码也一样，正常情况有2种可能，一种是中2个号码亏损6元，一种是中3个号码盈利6元，出现不中式中全的 情况较少，但坐3望4的希望仍然有。

　　以上单双分类大致告诉你以下信息：

　　1、大大单单、大大双双和小小单单在各包含3组任选二的前提下，平均2期各类可弹出1组任选二。

　　2、小小双双势单力薄，平均间隔5~6期才可"重温旧梦",如果短期内频频出击，一旦冬天来临，尽是门庭冷若。
