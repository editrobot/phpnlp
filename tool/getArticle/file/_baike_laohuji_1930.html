老虎机规则(Slots) ( 乐透游戏 )

作者: 博之道 来源: 未知 时间: 2011-11-21 阅读: 次 老虎机的秘密

　　老虎机游戏是一个很受欢迎的一种游戏。

　　大约是在1896年,一个叫Charles Fey 的男士在旧金山成功制造第一台商业用途的老虎机。 其机身由铸铁制造并带有3个内部卷轴,一个投币口,以及启动机器用的外部把手。它很快就扎根在沙龙,赌博场所和许多零售商店。

　　每个卷轴上有一些图案(多为10 到 20–不同的机器差别很大),拉动拉杆可代动卷轴。卷轴停转后,各卷轴将会随机在窗口显示图案。根据不同的图案组合,玩家赢得不同的奖金–通常是多倍于下注金额的奖金。

　　开始游戏：

　　玩吃角子老虎机时，只需将筹码投进投币口，然后拉动手柄(点击手柄)或单击“Spin”按钮转动卷轴。卷轴停转后,各卷轴上的图案将会随机停在"游戏线(Play Line)"。根据游戏线上不同的图案组合,玩家赢得不同的奖金。(老虎机面屏上通常有奖金或硬币结存显示器)。

　　您可以将任意多个筹码放进投币口，每一部老虎机都可以被设置为：$0。25 老虎机，$1 老虎机，$5 老虎机。如您把 $10 赌城筹码存到设置为 $1 老虎机，老虎机内硬币结存数为10。如果您把老虎机设置值改为$0。25，老虎机内的硬币结存数为40(筹码总额为 $10)。

　　每次游戏(拉一次把手或旋转一次)可以赌一至三个硬币。每次赌多少个硬币请看COINS BET显示器。但是默认设置为每次赌一个硬币。您要一次赌两个硬币，请按一下BET ONE增加赌注。(若您再按一下, 变成每次赌三个硬币。继续再按一次，又变回每次赌一个硬币)。您可以直接按BET MAX 每次赌三个硬币。一旦设好每次赌的硬币数后，机器将一直沿用此设置直到改变设置为止。

　　奖金表通常就在老虎机上。本赌城有多种老虎机游戏。奖金表的各行有各图案组合的奖励数，竖列表示与每次玩多少个硬币相对应的奖励数。每种图案组合的奖励数有3种情况，根据您每次玩多少个硬币而定。每次玩多少个硬币，奖金表上与其相对应的竖列就会变亮。

　　每次游戏后，您可以按机器上的COINS OUT(退钱)钮把老虎机内的硬币换成赌城的筹码，其等额资金将会增加到您的赌城帐号。

　　每种老虎机的游戏节奏都非常明快，将为您带来强烈的刺激和震撼！
