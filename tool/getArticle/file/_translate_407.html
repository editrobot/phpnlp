博彩业自古有英雄

作者: 博之道 来源: 未知 时间: 2011-09-23 阅读: 次 博彩译文

　　在中国的古文化历史里就有诗云：银牌高悬门市东，百万居然一掷中。谁向风尘劳斗色，博徒自古有英雄。这是清代爱国诗人丘逢甲，他在《岭云海日楼诗抄》中，形象地描写了澳门的开赌盛况的诗，诗中很生动写出了澳门在博彩业的繁华盛况：至高的银牌高高地悬挂在城门市区的东方，而在这众多人的博彩中居然一下子就掷中了这赢家的头彩。风尘世间里有谁在这场博彩中斗尽了自己的神采，博彩事业是自古英雄人才早已有的。

　　博彩在中国有着非常悠久的历史，博彩活动作为一种与娱乐共生的个人行为，其历史源远流长。应该说与人类的其他文明和习俗同步发展。而博彩作为一种产业，则是农耕文化出现以后的事情，是伴随着阶级分化和财富分配不均而发展起来的一种特殊现象。

　　早在周朝时期，已经有了文字史料记录了。在拜金主义思潮的影响下，有些人为了更快地拥有财富，当他们无法通过正当途径满足自己欲望的时候，就开始冒险走上博彩这条“发财之路”。

　　唐宋是中国博彩发展史上至关重要的时期。在博彩器具，博彩形式方面，唐宋融古贯今，颇有创制。这一时期的博彩形式有船子戏，采选叶子戏、屋架、双陆与长行，订马以及宣和牌等。其中创制于唐，演化至宋宣和年间产生出另一新品种骨牌--宣和牌。这种骨制赌具，对后世产生深远影响。骰子作为掷具一直沿用至今。牌戏，尤其是宣和牌的斗牌法直启后来“天九”、“牌九”、“接龙”诸法。“采选”、“升宫图”之雏形，其叶子戏的型制对“马吊”牌产生影响。而马吊牌与宣和骨牌杂交演化的结果又产生了中国的国戏“麻将 ”。

　　随着历史的一路发展下来，我们应该可以想到，至于元清朝时期博彩业，随着经济的发展已经 进入到一定的繁荣程度，资本经济在我国发展已经有了初步的雏形，人们的生活充足，所谓“饱而思娱”而博彩业无疑是一个很有激情并且对资本经济发展极具推动作用的行业。但是博彩业真正在中国有如上面诗所说到的盛况，还得是清末时期。清末年，由于历史原因，澳门这个葡萄牙的前殖民地，近年来堂而皇之地把博彩娱乐业列为它的龙头产业，期望以现代化和国际化的博彩业为动力，带动其他服务业的协调和同步发展。

　　博彩业，很显然地在我国的社会历史中曾带动着古代经济的发展，而那些在博彩业里挥撒着他们人生的光芒的英雄们，他们不仅在他们的人生事业里找到了拼搏的激情，更如弄潮儿般在历史的海洋里推动着历史在惊涛海浪中前进！
