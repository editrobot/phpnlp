吉米巴菲特历尽艰难赢得内华达州博彩经营许可证

　　几十年来，吉米巴菲特一直用悠闲的音乐来回报他的听众。这位创作型歌手利用自己的名人效应在过去的十年里成功的经营了玛格丽塔连锁餐馆。现在，流行歌手将把他的标记印在美国的赌博之都。

　　上周，巴菲特获得了一张由内华达博彩委员会批准的博彩经营许可证。委员会经过不懈努力、统一了内部的不同意见，确保巴菲特获得许可的资格。虽然巴菲特有小小不然的官司缠身，但这些并没有阻止65岁的他获得许可。

　　去年年底，巴菲特在火烈鸟酒店的第一家赌场-玛格丽塔开张，之前巴菲特在这里经营了一家餐厅。博彩业将成为扩展他备受欢迎的玛格丽塔品牌的下一步。

　　巴菲特的一生创作的音乐都是以海滩的声音为源泉，他在娱乐领域营造的独特氛围、海滩悠然迷的狂热崇拜只有“感恩而死乐队”和“费西合唱团”能与之媲美，键盘是他实现人生的地方，是玛格丽塔品牌的开始

　　硬石摇滚是博彩业和娱乐业联姻的成功典范，在经典摇滚和摇滚纪念品上享有盛誉后，硬石在包括拉斯维加斯的赌场、好莱坞和坦帕继续拓展他们的品牌，巴菲特希望玛格丽塔可以效法硬石的发展模式。
