内华达州州长宣布新的博彩政策委员会

作者: 博之道 来源: 未知 时间: 2011-11-10 阅读: 次 博彩译文

　　自内华达州博彩政策委员会在20世纪80年代成立以来，它一直作为内华达州国会议员的赌博顾问。

　　在线赌博行业的合法性一直受到质疑，即使它为国家带来了很高的财政税收。值得庆幸的是，自从4月15日的“黑色星期五”事件在美国蔓延，越来越多的州在认真考虑一个计划，以法律充分规范网上赌博。

　　而最近的事态发展带来越来越多好消息，特别是在赌博的监管政策方面。这一消息来自美国的内华达州。内华达州与拉斯维加斯被认为是世界上的拥有最多赌博资金的赌博场所，而且一直是这个行业受关注的焦点。

　　最近，内华达州总督宣布，他打算继续在内华达州发展网上赌博行业，并实行有效的监管维护政策。为了达到此目的，他重新设置内华达州博彩政策委员会机构，他本人也是该机构的领导成员之一。新的博彩政策委员会机构的作用是为网上赌博行业的立法者和监管者提供相关的法律政策咨询。

　　在上周五举行的一个会议上，内华达州州长说，新的博彩政策委员会将引领在线赌博行业走进现代的赌博行业。内华达州官员布赖恩桑多瓦尔说，新的博彩政策委员会将为在线赌博行业规划一个发展前景。他还认为，内达华州州长作为一个地区领导人，为确保内华达州在线赌博和博彩娱乐的地位做出了不少贡献。

　　博彩政策委员会第一次成立是在1961年时，当时的总督批准设立这样一个机构，目的正是为了规范赌博行业。而桑多瓦尔在接受记者采访时透露，新的博彩政策委员会自成立以来表现出前所未有的积极。

　　目前，新的博彩政策委员会将研究网上赌博的各种问题，包括网上赌博合法化的法律问题。他们计划制定一些互联网赌博的法律政策。
