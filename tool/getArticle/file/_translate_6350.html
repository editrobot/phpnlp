欧洲杯热身赛―自信的克罗地亚轻取爱沙尼亚

作者: 博之道 来源: 时间: 2012-05-28 阅读: 次 博彩译文

　　克罗地亚队3-1轻取爱沙尼亚，用完美的胜利回报了普拉Aldo Drosina体育场的主场球迷。

　　韦德兰·乔尔卢卡和尼古拉·卡利尼奇在开场20分钟就帮助主队2球领先，而奥格宁-武科耶维奇锦上添花打入第三球。客队的康斯坦丁.瓦西里耶夫打入安慰性的一球。

　　昨天，主帅比力奇就宣称比赛会有一个令人意想不到的阵容，他兑现了他的承诺，给许多球员机会来展示他们值得进入他的最终大名单。

　　克罗地亚队仅用了15分钟就确定了他们的优势。爱沙尼亚队没有处理好右翼的传球，韦德兰·乔尔卢卡近距离打入一球。4分钟后，克罗地亚2-0领先，这个结果提升了主帅的信心。爱德华多脚后跟妙传左侧的伊万·佩里西奇，后者犀利的传球给到了尼古拉·卡利尼奇，冷静地完成射门。

　　克罗地亚下半场继续攻势，但直到第81分钟才扩大了比分优势。两位替补球员配合完成了得分。奥格宁-武科耶维奇接到塞姆.福萨里科的精妙传球完成射门。爱沙尼亚的康斯坦丁.瓦西里耶夫通过任意球破门帮球队挽回颜面。但是克罗地亚的阵容已经显现了他们不俗的实力。

　　比力奇的球队明天将启程去奥地利的Bad Tatzmannsdorf进行欧洲杯最后一周的备战。6月2日他们会在奥斯陆和挪威打夜场友谊赛，之后的6月10日他们就会在波兰波兹南迎来C组的第一场比赛，对手是爱尔兰队。
