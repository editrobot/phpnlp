2011年NBA总冠军投注

作者: 博之道 来源: 未知 时间: 2011-10-01 阅读: 次 博彩译文

　　迈阿密已用推土机推平的方式进入决赛，赢得4场比赛之后直接粉碎头号种子公牛队。热火队一直占据主导地位，包括NBA的4连败赔率。
　　但是，大约是以前韦德的失败提醒了达拉斯，热火队在2011年最引人注目的是勒布朗 、詹姆斯和波什。詹姆斯已经在季后赛引起了轰动，平均每场26.0分，8.9个篮板，5.5次助攻，1.7次抢断和1.5次盖帽。这甚至并不是他最出色的防守。
　　韦德依然是被受关注的，尽管詹姆斯已经超越了他。作为专营权的事实领导者，韦德仍是热火队季后赛的希望，或者至少你会认为他最是关键的球员。韦德在与公牛队的对抗赛中没有得到良好的发挥，每场比赛场均只有18.8分，低于前三场赛事中的20。在第五场比赛中，韦德被看见坐在板凳上护理他受伤的肩膀。韦德在热火的表现欠佳，但最后还是赢了公牛队。
　　达拉斯在短短的五场比赛中，完成了年轻有为的俄克拉荷马城雷霆，包括关闭了该系列的三连胜。公牛队已经令人难以置信的被列在了季后赛NBA的赔率名单中。
　　诺维茨基在2011年里犹如处在风暴眼之中。一旦贴上了季后赛的颈饰标签，诺维茨基就需要走很长的路才能揭下标签。他在德国的赛事中平均每场28.4分和7.5个篮板，同时投篮命中率51.7％。如今他对自己的比赛提出了更高的要求，得分40分以上，平均每场32.2分，投篮命中率55.6％，甚至比这更高。
　　特里是目前唯一的特立独行的人。在2006年与迈阿密的决赛中有平均每场比赛17.3分的成绩。特里的特别，是因为他偶而有爆发性的表现。
　　在对阵雷霆队的比赛中，真正令人惊讶的是马里昂，他有几次演变成了“黑客帝国”，得到了18分和26点的爆发。
　　迈阿密和达拉斯两次在常规赛中相遇。公牛队在11月106-95主场取胜，并在12月98-96客场，总数超过了两次。
