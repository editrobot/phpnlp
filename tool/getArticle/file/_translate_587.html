　　欧洲扑克巡回赛已经在伦敦取得圆满结束，8天的现场锦标赛扑克行动就此告一段落。

　　有31事件中有五个最终买入25万英镑，这些主要事件都是跨时间表跨区域的。

　　将会有一个众所周知的有趣的扑克面孔组合，以及一些在线扑克冒险成功人士的面孔出现在大众面前。

　　有两个场馆被定在希尔顿伦敦Metropole酒店和维多利亚格罗夫纳赌场，以分担欧洲扑克巡回赛的责任。

　　英国首都伦敦将照顾了大部分的比赛份额。

　　这项设计在伦敦扑克节9月中旬以来一直运行。

　　你可以期望有些玩家在节日期间采取行动，包括贾森名士，乔纳森杜哈明，贝特朗Grospelier，乔卡达，鲍里斯贝克尔，尤布洛姆。

　　伦敦的主要活动还包括横跨英国和爱尔兰扑克巡回赛的最后一场比赛。

　　排名前三名的玩家在比赛结束后将获得的奖励是UKIPT在2012年的主要活动与季票。

　　此外，排名前100位的玩家将获得一个特殊的？ 10,00077.50点EPT伦敦主赛事奖励。
