美国堪萨斯州城镇居民团结一致打击赌瘾

作者: 博之道 来源: 时间: 2012-03-03 阅读: 次 博彩译文

　　尽管赌博可以为当地带来数亿美元的财政收入，但赌博的广泛流行会产生一些消极的方面，最消极的方面是问题赌博的概率不断升高。美国的堪萨斯州城镇开辟了国有赌场，城镇居民团结一致打击赌瘾。

　　赌场不是使人们产生赌瘾的唯一原因，赌徒的自控能力不强也是原因之一。所以一些城镇已经开始建设自己的问题赌博工作队。在堪萨斯州，问题赌博工作队正在创建提高居民赌博意识和处理赌瘾的治疗方案。

　　堪萨斯州居民马库斯·摩尔说：“随着近年来赌徒不断增加赌博金额，这已经成为一个必要的措施，以避免普遍的赌博成瘾问题。我常常可以看到一些人无比沉溺于老虎机的世界，玩老虎机赌钱游戏已经成为了他们的乐趣，但他们这样的行为太危险了，他们需要知道自己的问题所在。”

　　经济衰退时期，堪萨斯州的选民同意在当地建设四个国有赌场，以缓解财政压力。从那时起，赌场相继开业。

　　为了加强当地居民对问题赌博的认识，当地社区领袖，民选官员和商界领袖都为此付出了巨大努力。社区领袖威尔逊说：“居民们团结起来一起打击赌博成瘾问题，这是一个很值得骄傲的事情。我相信，治疗和教育是击败赌瘾的关键。及时有赌博存在，我们的社会也会是美好的。”
