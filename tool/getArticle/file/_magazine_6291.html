2012世界扑克锦标赛前瞻

作者: 詹一帆 来源: 《环球博彩》 时间: 2012-05-24 阅读: 次 博彩杂志

　　世界扑克锦标赛（WSOP）无疑是世界上最大型的扑克赛事，今年五月下旬将再次拉开帷幕。尽管仍基本沿袭往年模式，本届在赛程上也有一些调整，包括一则有可能改变扑克面貌的大新闻。

　　无论扑克界发生什么，在可预见的未来内，每年在拉斯维加斯举行的WSOP作为最大型扑克赛事的地位将坚不可动。不仅因为它的名称，其背后极具标志性的历史也将它与其它赛事区分开来。它持续近两个月，世界一流选手云集这里，共同角逐金手链以及世界冠军的美名。

　　2004年，哈拉斯娱乐（Harrah's）接替比尼恩（Binion's）开始主办WSOP，今年将再次负责该赛事，地点仍是位于拉斯维加斯大道向西一英里的利澳全套房娱乐场。过去几年中不断听到有关赛事选址的批评，但哈拉斯拒绝向压力低头，始终没有动摇。人们常问为什么不迁至同为哈拉斯旗下物业的凯撒皇宫，场地更宏伟，且位于拉斯维加斯大道的正中心。

　　好消息是今年将增加近百张牌桌，由去年的378张增至470张。这将令所有人感到欣喜。选手们被换桌的次数会减少，工作人员在监管时也会有更多灵活性。

　　牌桌的增加意味着主赛事首日比赛天数的减少，将由通常的四天减少至三天。这是一个积极的举措，缩短了主赛事持续的时间。

　　主赛事赛日1a、1b和1c将分别于7月7、8和9日举行。1a和1b晋级的选手将合并为7月10日赛日2a的阵容。1c 晋级的选手将进入7月11日的赛日2b中。之后所有幸存选手于7月12日一同进入第三日比赛，直到7月16日诞生出最后九名选手。

　　另一项主要调整是"十一九"会战改成了"十月九"，因为它与十一月的美国总统大选撞期。九位决战选手仍是休战 数月后重返沙场，尽管具体日期有所变动，但这种独特的安排保持不变。扑克界对此看法不一。我的立场较为居中，它有好处也有弊端。我并不反对如此安排，只要这是每年唯一一场采用此结构的赛事。

　　WSOP一直坚持提供一些一万美元买入的一次性"世界锦标 赛"赛事，涵盖多数扑克形式。这是一种很好的创新，为大玩家提供了小阵容和强对手的顶级赛事。"扑克选手锦标赛"今年继续，买入费仍为五万美元。它持续五天，被很多人认 为是非官方的世界扑克锦标赛。

　　今年的赛程中包含了史上买入 最高的比赛The Big One for One Drop，买入费一百万美元，承诺将成为有史以来最引人瞩目的比赛。有关举办这场赛事的讨论已进行了若干年，问题不在于是否举行，而是何时举行。这个"何时"已定为2012年7月1日，注册上限为48人。去年12月5日凯撒皇宫公布已有22名选手承诺参加。截止出版时，他们宣布已有30位左右选手确认参赛。参赛人数很可能会接近48，因为很多豪客选手将联合起来买进。同时我想一些商业巨头也不会吝于甩出这笔"小菜一碟"的本金，坐下来与世界顶级选手对决。胜出者将获得有史以来首条WSOP白金手链。

　　目前为止，为此赛事注册的选手包括太阳马戏团的创始人盖·拉里贝特，以及多位世界顶级（和/或最富有的）选手：

　　每张入场券将捐赠111,111美元给One Drop，一个致力于为贫困地区解决清洁水的慈善机构。目前为止，他们在海地和尼加拉瓜等地做了大量出色的工作。

　　从扑克锦标赛的角度来看，这场比赛引出了一系列来自选手和主办方的有趣问题。会有多少选手参与？这只是一次性比赛，还是会成为每年的固定安排？扑克离它的起点是否走得过远？这些超级豪客赛是否令扑克失去了其核心意义？尽管这种超级豪客赛遇到一些负面言辞，毫无疑问这场比赛将吸引大量关注。我们会在7月1日比赛展开后在我们的网站wgm8.com上及时报道赛况。

　　我认为，总体来讲哈拉斯为第43届WSOP作的安排已非常到位。尽管我也认为改换场地将令他们受益，但显然哈拉斯认为没有必要。他们不断组织出令众选手满意的高质量比赛。因此，预订好去拉斯维加斯的机票，与WGM一起加入到这场扑克盛事中。与往年一样，一定会涌现出高水准的亚裔选手，但2012年主赛事上会否出现一位亚裔选手一路过关斩将，坚持到最后？希望如此！
