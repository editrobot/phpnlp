　　据介绍，四年来，迈克·斯奈德一直保持着每次购买一张彩票的习惯，但2011年11月18日他决定花20美元购买两张彩票，无巧不成书，正是这关键的第二张彩票让他中得100万美元大奖，和妻子变成了百万富翁。迈克说，“我做梦也没想到能中100万美元的。当时我觉得即使我能中些小奖也会非常高兴的。”他说，他一直期待能中5000美元或者25000美元，没想到会中这么多，他很高兴自己买了两张彩票，这是一个回报很高的决定。

　　目前，李·斯奈德从事的是与医疗健康相关的工作，她说，她爱自己的工作，不打算离职。她的丈夫迈克已经退休，迈克说，这笔意外之财将使自己的退休生活过得更幸福一些，他会节省这些钱，理智地使用这些钱。
