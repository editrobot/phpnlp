世界十大赌王简介分析

作者: simo 来源: 时间: 2013-02-26 阅读: 次 赌王介绍

　　赌王这个称呼并不是随便就能封上的，这需要一个赌客在娱乐城中有足够的成绩，在世界各地能称得上赌王的人并不多，最为人知的十位赌王更是人中之龙，现在一起了解一下他们的事迹。

　　第一、22岁赌王”豪取915万美元奖金

　　在美国内达华州赌城拉斯韦加斯举行的世界扑克大赛中，22岁的丹麦选手彼得-伊斯特盖特取得最终胜利，赢得915万美元，成为历史上最年轻的“赌王”。

　　第二、俄罗斯扑克高手伊万·杰米多夫

　　2008年11月12日的拉斯维加斯世界扑克大赛年度总决赛中，俄罗斯选手杰米多夫在决赛中对阵丹麦选手伊斯特盖特，最终败北于新晋“赌王”获得第二名，也获得了580万美元的奖金。

　　第三、美国赌神菲尔·赫尔穆特

　　素有“扑克顽童”之称的美国选手菲尔·赫尔穆特是扑克牌界的传奇人物，他曾11次在世界扑克冠军联赛中夺冠，这一成绩至今无人能及。赫尔穆特现年43岁，出生在美国威斯康星州麦迪逊市，25岁时便以一名青涩新手身份，摘得职业生涯中的第一个世界冠军。之后，赫尔穆特曾20余次出现在世界扑克冠军联赛的决赛牌桌上，赢得奖金总额超过400万美元，并成为电视节目《名人扑克对决赛》的嘉宾主持人。赫尔穆特现在与妻子和两个儿子居住在加利福尼亚州帕洛阿尔托市，他的妻子是一名心理学家。“扑克顽童”的绰号源于赫尔穆特的火爆脾气。一旦在牌桌上成为输家，他就会大发雷霆。

　　第四、亚裔选手获扑克大赛冠军获800万美元

　　2007年7月18日，39岁的心理学家杰里米·扬获得了世界扑克系列大赛冠军，拿到了825万美元奖金。13岁到美国的老挝人杰里·扬说，他只是半职业选手，两年前开始参加扑克比赛。

　　第五、赌王车敏洙创造传奇经历

　　他是“赌城”拉斯维加斯历史上收入位居前三位的职业牌手。目前，以车敏洙真实故事改编的电视连续剧《ALLIN》在韩国播出后，引起了巨大轰动。
