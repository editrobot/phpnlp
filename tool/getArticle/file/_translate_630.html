大西洋城的赌场因游客人数减少而下跌

作者: 博之道 来源: 未知 时间: 2011-10-10 阅读: 次 博彩译文

　　事实上，美国的赌博行业是红红火火的，特别是网上赌博。也许正是因为网上赌博行业的欣欣向荣，美国的第二大赌场——大西洋城赌场游客人数下降了6％。

　　在2010年，超过25万人次前往大西洋城赌场，但仍下降6％左右。对大西洋赌城来说，唯一的好消息是，这些都是不太活跃的客户，证明大西洋赌场依然有魅力吸引新客源。

　　大西洋城的赌场赌博经济遭受到周边国家快速增长的经济所冲击。

　　在过去的五年里，大西洋城一直试图使自己成为像拉斯维加斯赌场那样成为高消费顾客常光顾的地方，能使顾客们在这里逗留几天，参加一些节目，去购物，在高档餐厅用餐，在度假胜地玩赌博游戏等等。

　　特朗普娱乐公司的首席执行官，鲍勃格里芬说，大西洋赌城下跌的趋势是明显的。这是他通过分析特朗普娱乐公司在大西洋城的经营状况得出来的结果。

　　“忠于大西洋城的玩家依然会在大西洋赌城下大注”他说， “我们正在失去的客户是那些被网上赌博吸引的赌徒，他们认为网上赌博更方便，因为他们只用呆在家里就可以了”。

　　调查发现，在2010年，约有27亿人参观了大西洋城的11个赌场，其中大多数是驾汽车而来的客人。巴士乘客人数下降了13％，小于3.8万。波洛克据此说到：“巴士将缓慢地与大西洋城赌场告别”。

　　在宾夕法尼亚州，纽约，特拉华州和康涅狄格州郊区的赌场，针对的主要客人是那些每天跳槽的赌徒，这些跳槽赌徒曾经是大西洋赌城的摇钱树。

　　2003年Borgata酒店赌场水疗中心开幕是一个典型的例子，这也是赌博行业发展的新战略。 Borgata重点是向高消费客户提供一个拉斯维加斯式的度假体验。
