进入英国赌场需要身份证

作者: 博之道 来源: 未知 时间: 2011-10-12 阅读: 次 博彩译文

　　OK - 这是一个有趣的问题，进入英国赌场是需要身份证验证。

　　在英国2005年10月之前，英国法律规定，该国的赌场作为一个俱乐部，需要登记他们的客户成员，并要求他们在成为会员之后需要等待24小时以后方可在赌场内玩任何游戏。该法于2005年10月1日被废除。现在，您可以在英国的赌场上马上发挥。

　　一些私人赌场俱乐部可能仍然需要您首先申请成为会员。

　　所有赌场的游客成为其成员的要求，从2007年12月被取消。你只需要注册您的姓名，出生日期，此外赌场必须记录你的照片。

　　您不再需要护照编号。如果你碰巧赢得金额，赌场将可能会需要提供身份证进行身份认证，然后才能想你支付奖金。此前，英国所有的赌场被视为俱乐部。如果你已经成为会员，你需要想提交护照或带照片的身份证。成员通常在24小时内批准。因此，新的系统，显然简单得多。

　　在英国，赌场是受英国博彩委员会管理的。在英国经营赌场必须符合一定的标准：他们必须提供各种各样的游戏，例如在老虎机、二十一点等，以及食物和饮料服务，并有足够的厕所设施。

　　赌场比赛一般包括轮盘，21点，骰子，百家乐，加勒比海扑克，以及当前流行的老虎机游戏及视频扑克游戏，您可能还会看到轮盘机，如果你喜欢玩轮盘赌博，大多数英国赌场提供欧洲轮盘。

　　在英国赌场如何着装

　　在英国赌场如何着装？对于这个问题，通常我们会说简单的便装，只要不是短裤和拖鞋就可以。

　　给小费，不给小费？

　　以前这是被禁止的，但现在英国赌场遵循一种更拉斯维加斯线。你可以提示，如果你喜欢。

　　如何在英国赌场投诉

　　如果你觉得你在英国的赌场中受到了不公平待遇，你可以在英国赌场进行投诉，并且有相当多的选项。

　　首先，当然需要和赌场服务员协调。无法解决时，再和然后再和楼层经理协调，最后和赌场经理协调。

　　如果你感觉这样很浪费时间，效率很低，也可以尝试和该公司的高级管理人员联系。

　　如果这一切都失败了，你申请赌场仲裁（IPCA）的独立调查小组来帮你忙：

　　IPCA，邮政信箱5698，伦敦，W1A 7AW。

　　该小组是由退休的资深成员，赌博委员会及其他独立的赌场业专家组成。小组成员名单可以在英国赌场协会看到。

　　IPCA将处理收到的书面纠纷直到他们完全满意。
