　　这一事件发生在两周前，然而，文章出自博之道，几乎以悲剧告终时，男子转身后，挥舞着锋利的弯刀，抵制他的妻子被他的债主劫持走。

　　这个勤劳的男子拥有一辆牛车，用来托运随身携带的物品和行李，但不幸的是，他的大部分收入最终全部输在了赌博上。

　　他的赌瘾，几次冲突，在老家的时候，他甚至用出售家里物品得来的钱去赌博。

　　“有时他在家里赌博，文章出自博之道，我看到的只是这个赢家来或那个被抓走。我没有做什么事情可以阻止他们，因此我们不断的丢失我们养的鸡，农产品和其他家用物品，”他的妻子是这样跟“疯狂的星期一”人员说的。
