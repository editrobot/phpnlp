　　当“伯爵振动”模式开启，你的iPhone或iPod将计算您所指定的，或打到您指定的值。你可以使用你的口袋里的iPod，并按下按钮，总计算卡记录。此iPhone的二十一点应用程序比人工方法更准确地计算二十一点卡数。您可以在最流行的卡计数方法之间做调整。
