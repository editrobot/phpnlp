　　牢记这一点。你不能改变它看中的制度。这是一个游戏的恒定规则，再加上你玩游戏物理特性。

　　一个良好的资金管理系统，帮助您控制您的风险的金额。这将使你玩更长，保持更多的利润，以缩短你的损失。

　　利润指标和损失限额

　　你应该总是设定一个盈利目标和损失限额。当你达到盈利目标或损失限额之后，你需要及时停止继续赌博游戏。你不应该中途改变损失限额，除非你有另外一个利润目标。“库存”的一些奖金，然后为您的资金设置的新限制。
