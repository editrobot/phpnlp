　　在世界各地，贵族每天都创造世界上最伟大的游戏体验。 2011年，在G2E，贵族将展现一个全新的令人振奋游戏类别阵容，全球公认的贵族数学模型由行之有效的操作系统驱动。

　　美洲贵族总统钦尼克说，我们推出了不同于任何行业已经出现过的新游戏种类和技术。 “我们在过去的几个星期一直在给客户展现技术，并得到了非常积极的响应。我们非常高兴G2E引进这些产品并将其推向世界。“

　　贵族的Viridian是主动卷轴技术相混合。是正在推出一个令人惊叹的游戏，可能从功能上看都在同一种5排步进基地游戏之上，这有全屏幕的视频和一个3排步进奖金，是一种新线游戏！神奇的是，可能与贵族创新的卷轴上窗户的百叶窗液晶相结合。

　　首先是Viridian的混合线路胜出，他推出三种游戏：波斯奖、精灵富贵和大溪地珍宝。此外，在该行的佐罗和佐罗传奇归来、面膜＆玫瑰？，三个层次的游戏都有5个不同的功能上的进步。此外，超旋是有一个奖金的银行免费游戏，推出渐进性和信贷奖奖金的新系列。

　　Stormin是最高奖金游戏，包含奖金特点，可作为非渐进式或作为独立的渐进配置。对于玩家的挑战在于要收集逐行或信贷奖励风暴符号。

　　贵族还推出了黄金级现金快车。根据以前广受欢迎的原现金快车，黄金类现金Express的功能和乘数轮功能在新的黄金级，高频率基地游戏，奖金和中奖的特点和五个层次上有了很大的进步。
