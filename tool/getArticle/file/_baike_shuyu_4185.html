双色球博彩术语介绍

作者: 博之道 来源: 时间: 2012-02-15 阅读: 次 赌博术语

　　权应数

　　权应数是根据上期的出球，通过一定的加权公式运算得出的一组对应数。所取权数可以是一组数，所取的权数不同，则所求出的对应数也会不同。这两个概念被称为双色球奇异数。

　　在这两类数的正负2的范围内，每期都会有号开出来。有时候还会不止一个。很多时候，开出来的号，就是它们本身。所以我们在这两类数的正负2的范围可以准确的找到胆。

　　码和反应值

　　码和反应值是近期各期开出奖号的和值，码和是能反应各期和值走势的值。

　　号码偏度

　　单注所有号码与上期开奖号码之差（以结果的绝对值为准）的最小值中的最大的一个。

　　单注所有号码与上期开奖号码之差（以结果的绝对值为准）的最小值中的最大的一个。

　　号码散度

　　单注所有号码与当前号码之差（以结果的绝对值为准）的最小值中的最大的一个。

　　旋转矩阵

　　指通过电脑手段将备选出来的号码进行组合的一种优化的数学组合方法，旋转矩阵是目前彩市较为流行和实用的一种号码组合方法。

　　聚集和远距

　　指当期中奖号码中号码最少3个或3个以上聚在一起，或相分开最近的间隔达6格以上的号码现象

　　旺数周期

　　指中奖号码在小区或跨越了小区的概念在某个区域连续3期以上出现6个号码以上的情况，称为旺数周期。

　　叠连和连叠码

　　指上期中奖号码出现后，引出下期1组相重叠的连码，或者上期出现连码后，引出下期其中1个号码重叠，就分别称为叠连码和连叠码。

　　衔接号码

　　指双色球红区中最近几期的号码能够相互按自然数顺序紧密衔接起来的号码，如1-2-3、4-5、6-7-8等。
