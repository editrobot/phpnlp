　　攻略一、取上期开奖号码中的5个号码，5个号码的平均值及它的+1或-1的号码。

　　例如：上期开奖号码是“01、02、03、04、05”，其和值是01+02+03+04+05="15，平均值为15/5=3，因此，本期的前一号码定胆为02、03、04。

　　攻略二、用18减去前2期前一号码之和，取差值和它的+1或-1的号码。

　　例如：前一期的前一开“01”，本期的前一开“10”，两期之和为01+10="11，用18-11=7，因此，下一期的前一号码锁定为06、07、08。

　　攻略三、取上期第1个号码与第5个号码的和与差。

　　例如：第1期—第5期的开奖号码分别是：第1期：03+07+01+08+04，第2期：07+02+11+08+01，第3期：08+06+02+05+03，第4期：11+05+06+01+10，第5期：01+07+10+08+03。这5期用此方法取得了连锁佳绩。其中，第1期的03+04是第2期的07，第2期的07+01是第3的08，第3期的08+03是第4期的11，第4期的11-10是5期的01。

　　攻略四、用上下期的前一号码进行加减，取和数与差值号码。

　　例如：前1期前一开“06”，本期前一开“01”，那么06+01="07，06-01=05，因此，下期前一号码投注05、07。
