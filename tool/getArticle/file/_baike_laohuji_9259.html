X战警老虎机玩法

作者: lkq 来源: 时间: 2012-11-14 阅读: 次 老虎机的秘密

　　X战警（X-men）老虎机游戏是一款电子游戏，X战警老虎机游戏玩法具体如下：

　　如何进行游戏

　　点击位于屏幕左下角的面值按钮“点击以更改”即可选择投注额面额。

　　点击“每条线的投注”以选择线注。每点击一次，线注即增加一个币。当达到最大值，再次点击即重设线注为一个币。

　　点击“线”以添加赔付线，或使用卷轴两边的数字按钮选择赔付线。您选择任何赔付线时，也同时选中所有低于此线的赔付线。例如，选择6条赔付线则意味着激活了1-5线。点击“全押”即激活所有赔付线，并且以最大额的线注旋转卷轴。

　　每轮游戏总投注额 = 线注 x 激活的赔付线。

　　点击“旋转”即按照当前线和线注开始转动卷轴。在卷轴旋转过程中，旋转的按钮将会变成停止，您可以点击停止结束旋转的画面并马上显示结果。

　　卷轴还可以通过“自动开始”功能转动。点击“自动开始”上方的+ 或 – 以选择连续旋转的次数。点击“自动开始”转动卷轴。在卷轴旋转过程中，自动开始的按钮将会变成停止，您可以点击停止结束自动旋转或直到选择的自动旋转次数全部结束。

　　彩金将根据奖金表计算。线上赢得 = 线注 x 奖金表所示的相应倍数。分散图案赢得 = 总投注 x 奖金表所示的相应倍数。奖金表可以通过 “信息”页面查看。

　　在一条赔付线上，只按照最高获胜组合进行派彩，但在不同赔付线上的彩金则累加。

　　如果旋转赢得了彩金，在条目里显示的会是累加后的彩金，您可以点击条目关闭显示项。

　　赔付线和总彩金都会显示在游戏窗口下方的黑色条上。

　　信息页面

　　点击信息打开介绍游戏中不同部分的页面。您可以点击右下方的箭头进行翻页。

　　赔付表页面显示了所有获胜组合。当您打开一个获胜旋转时，获胜组合（数量和赔付率）会标示出并且不停闪烁。

　　X特别页面显示了赢得X-特别奖金需要满足的条件。

　　免费游戏屏幕有一个关于免费游戏中英雄和反面人物模式的概述。进入英雄模式或者反面人物模式页面，了解关于两种模式的详细说明。

　　英雄模式屏幕解释了免费游戏中英雄模式的规则。反面人物模式屏幕解释了免费游戏中反面人物模式的规则。

　　神奇多层神秘累积彩池页面会介绍如何激活并进入彩池并赢得四个彩池中的其中一项奖金，以及具体游戏的规则说明。

　　点击赔付表页面的“显示赔付线”按钮查看所有可能产生的赔付组合。点击“隐藏赔付线”按钮关闭此屏幕并回到奖金表页面。

　　点击返回按钮关闭信息页面并返回游戏主页面。

　　赔付线

　　激活的赔付线会在卷轴上标示。您可以点击“线”按钮激活赔付线并且查看它们的形状。

　　只有获胜组合出现在激活的赔付线上（一条您投注了金额的赔付线）才会赢 。此规则不适用于分散图案。 请阅读下文以了解有关分散图案的更多信息。

　　线注与您在该局的总投注金是不同的，线注显示的是您在每条赔付线上投注的金额。总投注是指您在该局游戏一共投注的金额。奖金表显示的赔付是以线注乘以赔付率，而不是总投注金。

　　关于赔付

　　赔付率在奖金表页面列出。要计算潜在彩金，只需用线注乘以赔付率即可。

　　如果同一条赔付线上出现两个获胜组合，则赔付彩金较高的组合。如果在多条激活的赔付线上获得获胜组合，奖金将会累加。

　　获胜组合需要是从卷轴最左边开始并且图案必须是连续的，否则无法赢得彩金。分散图案除外，关于分散图案的更多信息，请参照下文的介绍。

　　分散图案

　　分散图案（X战警标志）并不一定需要出现在特定的赔付线上而被视为获胜组合。如果出现两个或以上的分散图案 ，您会获得额外的彩金。分散图案赔付是以总投注相乘。

　　如果三个或者更多的分散图案同时出现在卷轴中，不管什么位置，您都将激活免费游戏。

　　百搭图案

　　百搭图案可以代替除分散图案外的其它任何图案，以组成彩金最高的获胜组合。根据赔付表规则，在一条激活的赔付线上，如果出现两个或者以上百搭图，那么您将获得额外的奖金。如果您所获得百搭图案的彩金比普通组合的彩金额大，那么您将获得这个百搭图案产生的额外彩金将替代普通的彩金。

　　免费游戏

　　在激活了免费游戏后，您首先要点击“点击以开始”按钮，进入免费游戏的屏幕。然后点击“继续”开始免费游戏。

　　免费游戏包括两种模式：英雄模式和反面人物模式。免费游戏会以英雄模式开始，英雄模式会以无限制次数进行免费旋转。反面人物模式包括8次免费旋转中。在英雄模式进行中，如果在第三条卷轴任何位置出现万磁王时，英雄模式停止并进入反面人物模式。

　　在反面人物模式中，当X教授出现在第三条卷轴的任何位置时，反面人物模式停止并进入英雄人物模式。反面人物模式的8次免费旋转会一直计算，也就是说，在整个免费游戏中，无论两种模式互相如何切换，反面人物模式中只能进行8次免费旋转。在坏蛋模式下第8次免费旋转中，如果“X教授”图案出现在转轴3上的任何位置，则将不会激活英雄模式。

　　在英雄模式，卷轴的图案只包含英雄图案、百搭图案和万磁王图案。万磁王图案将被视为一个额外的百搭图案。理论上说，如果在英雄模式中一直没有在第三条卷轴，您可以一直进行无限次数的免费旋转。

　　在反面人物模式中，卷轴的图案只有反面人物图案、百搭图案和X教授图案。X教授图案将被视为一个额外的百搭图案。每次旋转后，免费旋转次数都会减少一次直到达到零为止，免费游戏即结束。

　　在免费游戏中，免费游戏中的线注、币码值及所选择的激活赔付线数量与激活免费游戏的那一次旋转相同。每次旋转以后，彩金条显示彩金。免费游戏彩金条显示当前免费游戏产生的累计彩金。

　　所有免费游戏完成以后，页面会显示所有的彩金。游戏彩金显示获得免费游戏那一把命中的彩金，特别彩金显示的是在免费游戏中赢得的彩金。总彩金就是把所有彩金（游戏彩金及特别彩金）累加。点击“继续”按钮回到主游戏卷轴。

　　X-特别奖金

　　当在第二、第三及第四条卷轴中如信息页面描述的五个固定位置都同时出现英雄图案，就可以激活X-特别奖金。

　　命中X-特别奖金时，玩家可以在赢得普通获胜组合的同时获得相当于总投注额五倍的彩金。X-特别奖金无法在免费游戏中激活。

　　神奇多层神秘累积彩池

　　神奇多层神秘累积彩池是一个多层次的累积彩池。此彩池可以在所有连接神奇老虎机游戏上累积。

　　激活并进入累积彩池的机会是完全随机产生的，将可能在彩池连接的任何游戏的任一旋转激活。这与旋转产生的图案无关，就算是没有命中任何获胜组合的旋转也有可能可以激活并进入累积彩池。当然，命中了获胜组合的旋转也可能激活累积彩池。激活彩池机会只受投注金额大小的影响，投注金额越大，激活彩池的机会就越大。

　　彩池一共有四个级别：能量彩池、增强能量彩池、超级能量彩池和终极能量彩池。不同级别的彩池分别包含不同数额的彩池金额。累积彩池游戏页面包含20个方格，您需要点击翻开方格，一旦您翻开三个同样的图案，就可以赢得相应的彩金。

　　神奇多层神秘累积彩池游戏包含一个计时器。如果您没有在规定的时间内点击方格，系统会随机为您选择方格。

　　按钮

　　自动开始/停止：

　　连续旋转卷轴数次。/结束自动旋转功能。

　　线：

　　增加一条激活的赔付线。

　　全押：

　　以最大线注激活所有赔付线并转动卷轴。

　　旋转/停止：

　　转动卷轴。/停止旋转画面马上显示结果。在自动旋转进行时，点击“停止”按钮停止旋转画面马上显示结果。点击自动开始模式中的“停止”，不会结束自动旋转模式。

　　“+”和“-”按钮：

　　增加或减少自动开始连续旋转的次数。

　　每条线的赌注

　　线注增加一个币码值。

　　信息：

　　打开介绍游戏各个部分的参考页面。
